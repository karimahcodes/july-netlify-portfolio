About Me

I'm passionate about using tech to build community and find creative new tools to meet the needs of marginalized groups.

Prior to koeco, I worked with behavioral healthcare, nonprofit and social change organizations. Throughout these experiences of working with diverse groups of people, across ages, abilities, and other social identity groups, I've learned to evaluate challenges from multiple perspectives to determine a viable solution. My preferred style of working on a technical project blends compassion, collaboration, and client-driven solutions with code. I can't wait to work with you!

Technologies and Skills

In addition to familiarity with no-code platforms like Wix and Squarespace, I have worked individually and collaboratively on projects using the following tools:

  • Git and Github
  • Visual Studio Code
  • HTML5, CSS3, and Sass
  • JavaScript, Node and React
  • Express, MySQL and Passport
  • JSON, REST APIs and web APIs
